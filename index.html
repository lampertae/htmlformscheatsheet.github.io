HTML Forms Reference

Tags

Tag Name Description
<form> Form Used to collect information from users and send that information somewhere else.
<input> Input Used to render input fields on a webpage. The type attribute determines the form the input field takes.
<textarea> Textarea Used to create text-box which can take multiple lines of input.
<select> Select Used to create a dropdown list.
<option> Option Used to create options for a dropdown menu.
<datalist> Datalist Used to create a search/autocomplete input box.

Attributes

Attribute Name Description
name Name Used in an input tag to create a key-value pair. The value set to the name attribute acts as they key.
max Max Used to set a maximum value for a given input field.
type Type Used to determine the way in which an input element is rendered on a webpage.
pattern Pattern Used in an input element with the type: text. Validates the user input against a regular expression upon submission of the form.
maxlength Maxlength Used to set a maximum value length for an input field.
minlength Minlength Used to set a minimum value length for an input field.
